## Review Checklist

Please check the following:
1.  **Correctness**: Does the code do what it's supposed to do?
2.  **Security**: Are there any security vulnerabilities (e.g., injection, secrets)?
3.  **Performance**: Are there any obvious performance bottlenecks?
4.  **Testing**: Are there tests for the new functionality?
5.  **Readability**: Is the code easy to understand?
